• 위의 애플릿은 이진 탐색 트리를 구현한 것입니다.
  • "랜덤입력" 버튼은 입력난에 난수를 채우기 위한 버튼입니다.
  • "삽입" 버튼은 이진 탐색 트리에 값을 삽입하고자 할때 사용합니다. 먼저 입력난에 숫자를 입력한 뒤에 삽입 버튼을 누릅니다. 입력난에 숫자를 입력하지 않으면 난수가 입력됩니다.
  • "삭제" 버튼은 이진 탐색 트리의 노드를 삭제하고자 할 때 사용합니다.
  • "검색" 버튼은 이진 탐색 트리에서 특정한 값을 탐색하고자 할 때  사용합니다. 먼저 입력난에 숫자를 입력한 후에 검색 버튼을 누릅니다.
  • "모두삭제" 버튼은 전체 이진 탐색 트리를 지웁니다.
