선택, 삽입, 버블, 킉 정렬

  • 정렬의 종류를 선택합니다.
  • 수행속도를 선택합니다.(숫자가 높아질수록 속도가 빨라집니다.)
  • 다음 버튼을 누르면 단계적으로 실행됩니다.
  • 정렬에 사용할 숫자의 개수를 선택합니다. 화면의 크기 제한으로 인하여 16까지로 제한됩니다.
  • 직접 배열에 숫자를 입력하고 싶은 경우에는 직접입력 버튼을 사용합니다. 숫자들을 스페이스가 분리하여 입력합니다.
    • 예를 들면 12 23 34 45 56 67 등과 같이 입력합니다.
  • 막대<->숫자 버튼을 누르면 숫자가 막대 모양으로 변환됩니다.
  • 시작 버튼을 누르면 알고리즘이 시작됩니다.
  • 알고리즘 수행 도중에 수행 속도를 다시 선택할 수 있습니다.
  • 알고리즘을 다시 수행시키고 싶을 때는 정지 버튼을 누르고 다시 시작 버튼을 누르면 됩니다.
