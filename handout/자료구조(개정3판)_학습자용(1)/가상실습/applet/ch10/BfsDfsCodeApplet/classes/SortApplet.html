그래프 탐색(깊이 우선, 너비 우선)

  • 그래프의 노드의 생성은 마우스를 더블 클릭합니다.
  • 간선은 간선의 값을 입력한 다음, 마우스로 정점 사이를 드래그하면 됩니다.
  • 간선을 지우려면 가중치 0의 값을 입력하고 정점 사이를 드래그 합니다.
  • 정점을 이동하려면 컨트롤 키를 누른 상태에서 마우스로 정점을 드래그 합니다.
  • "연산의 종류"를 선택합니다.
  • 수행속도를 선택합니다.(숫자가 높아질수록 속도가 빨라집니다.)
  •  “시작” 버튼을 누르면 연산이 시작됩니다. 시작 버튼을 누를때마다 자동으로 랜덤한 숫자가 생성됩니다.
  • “다음” 버튼을 누를 때마다 단계적으로 실행됩니다.
  • 알고리즘 수행 도중에 수행 속도를 다시 선택할 수 있습니다.
  • 알고리즘을 중지하였다가 수행시키고 싶을 때는 “중지” 버튼을 누르고 다시 “시작” 버튼을 누르면 됩니다.
